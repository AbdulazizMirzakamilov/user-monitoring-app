# 📊 Система мониторинга активности пользователей

![WPF](https://img.shields.io/badge/UI-WPF-blue?style=for-the-badge&logo=dotnet)
![PostgreSQL](https://img.shields.io/badge/DB-PostgreSQL-336791?style=for-the-badge&logo=postgresql)
![C#](https://img.shields.io/badge/Language-C%23-green?style=for-the-badge&logo=csharp)

Настольное приложение для мониторинга и анализа активности пользователей. Система позволяет выявлять подозрительные действия, анализировать рабочее время и контролировать безопасность входов.

---

## ✨ Основной функционал

* **Аномальная активность** - автоматическое определение пользователей, превысивших установленный порог запросов.
* **Непрерывная работа** - выявление сотрудников, работающих длительное время без пауз (анализ сессий).
* **Анализ IP-адресов** - детекция входов с различных IP для предотвращения кражи учетных записей.
* **Умный поиск** - мгновенная фильтрация данных по ФИО или логину.
* **Валидация** - встроенная система проверки корректности ввода данных.

## 🧠 Реализованные отчёты

### 1. Аномальная активность
Подсчитывает количество запросов за период. Позволяет задать "порог" и увидеть только тех, кто его превысил. Сортировка идет по убыванию интенсивности.

### 2. Непрерывная работа
Использует **оконные функции SQL** для вычисления цепочек последовательных дней активности. Помогает найти самых трудолюбивых (или ботов).

### 3. Анализ IP
Агрегирует данные по количеству уникальных адресов. Если адресов много - система пометит активность как подозрительную.
